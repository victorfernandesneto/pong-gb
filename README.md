# Pofesso's PONG

## Description

This is the sourcecode of a reiteration (?) of the Atari's classic Pong for the original GameBoy, written in C, using GBDK library. This is my first GameBoy game, so I chose the most basic game that I could think of. It's a rather obvious choice to make a first game, but I learned a lot from it, mostly that the GameBoy is colorblind and can be a pain to draw simples sprites (such as the brackets).

## How to play

If you want to play the game, you can download the .gb file from the releases and run in any emulator. Alternatively, you can check the [itch.io](https://pofesso.itch.io/pong) and download from there.

### Enjoy?
